## 🔍 Project Overview

This project analyzes **1,000 supermarket transactions** across 3 branches to answer real business questions:

- Which product category generates the most revenue?
- How do customers prefer to pay?
- What does the daily sales trend look like over 3 months?

These are exactly the questions small business owners need answered to make smarter decisions.

## 🚀 How to Run This Project

### 1. Clone the repository
```bash
git clone https://github.com/flywithkhan29/sales-dashboard.git
cd sales-dashboard
```

### 2. Install dependencies
```bash
pip install pandas matplotlib seaborn jupyter
```

### 3. Add the dataset
Download the dataset and save it as `sales.csv` in the project folder:
```
https://raw.githubusercontent.com/dsrscientist/dataset1/master/supermarket_sales.csv
```

### 4. Open the notebook
```bash
python -m jupyter notebook
```
Then open `sales_analysis.ipynb` and run all cells.
